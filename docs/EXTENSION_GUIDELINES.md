# 🚀 নোশন ক্লোন এক্সটেনশন গাইডলাইন (একদম সহজ গাইড - Amateur Friendly)

আপনি কি আপনার নোশন ক্লোনে নতুন কোনো জাদু (Magic) যোগ করতে চান? তাহলে আপনি সঠিক জায়গায় আছেন! এই গাইডটি আপনাকে শেখাবে কিভাবে কোনো জটিল কোডিং না জেনেও মাত্র ৫ মিনিটে একটি এক্সটেনশন বানিয়ে ফেলা যায়।

---

## ১. খুব সহজ ভাষায়: এক্সটেনশন কী?
মনে করুন, নোশন ক্লোন অ্যাপটি হলো একটি **গাড়ি**। আর এক্সটেনশন হলো সেই গাড়িতে আপনার লাগানো একটি **এক্সট্রা স্পিকার** বা **নতুন হর্ন**। আপনি গাড়িটি না কেটেই জাস্ট নতুন কিছু প্লাগ-ইন করছেন।

আমাদের সিস্টেমে একটি এক্সটেনশন বানাতে মাত্র ২টো ফাইল লাগে:
1. **manifest.json**: এটা হলো আপনার এক্সটেনশনের "পরিচয়পত্র" (নাম কী, ভার্সন কত ইত্যাদি)।
2. **index.js**: এটা হলো আসল "লজিক" (সে কী কাজ করবে)।

---

## ২. ধাপে ধাপে আপনার প্রথম এক্সটেনশন তৈরি করুন

### ধাপ ১: একটি ফোল্ডার বানান
আপনার কম্পিউটারে একটি ফোল্ডার বানান, নাম দিন `my-first-extension` ।

### ধাপ ২: পরিচয়পত্র (manifest.json) তৈরি করুন
ফোল্ডারের ভেতরে `manifest.json` নামে একটি ফাইল বানান এবং এই কোডটুকু কপি করে বসিয়ে দিন:

```json
{
  "id": "hello-world-id",
  "name": "আমার প্রথম টুল",
  "version": "1.0.0",
  "author": "আপনার নাম",
  "description": "এই টুলটি সাইডবারে একটি বাটন যোগ করবে।",
  "icon": "🎈"
}
```

### ধাপ ৩: কোড (index.js) লিখুন
একই ফোল্ডারের ভেতরে `index.js` নামে আরেকটি ফাইল বানান এবং এই কোডটুকু কপি-পেস্ট করুন:

```javascript
/* 
  এখানে 'api' হলো একটি জাদুর বাক্স। 
  এই বাক্সের ভেতর থেকে আপনি অ্যাপের সব ক্ষমতা পাবেন।
*/

export const activate = (api) => {
  console.log("অভিনন্দন! আপনার এক্সটেনশন চালু হয়েছে।");

  // সাইডবারে একটি বাটন যোগ করার সহজ উপায়:
  api.ui.registerSidebarItem({
    id: 'my-button',
    label: 'ক্লিক করুন',
    icon: '🔥',
    onClick: () => {
      // ক্লিক করলে একটি মেসেজ দেখাবে
      api.notify("কি খবর? আমি এক্সটেনশন থেকে বলছি!", "success");
    }
  });
};

/* যখন এক্সটেনশন ডিলিট করবেন, তখন এই ফাংশনটি সবকিছু পরিষ্কার করে দিবে */
export const deactivate = (api) => {
  console.log("এক্সটেনশনটি বন্ধ করা হয়েছে।");
};
```

### ধাপ ৪: জিপ (ZIP) করুন এবং আপলোড দিন
এখন এই `manifest.json` এবং `index.js` ফাইল দুটোকে সিলেক্ট করে একটি **ZIP** ফাইল বানান। 
এরপর অ্যাপের সাইডবার থেকে **"এক্সটেনশন যোগ করুন"** বাটনে ক্লিক করে জিপটি আপলোড দিলেই সব রেডি!

---

## ৩. গুরুত্বপূর্ণ কিছু নিয়ম (সহজ নিয়ম)

1. **নামের মিল:** জিপ ফাইলের ভেতরে ফাইল দুটোর নাম অবশ্যই `manifest.json` এবং `index.js` হতে হবে। নাহলে অ্যাপ চিনতে পারবে না।
2. **পরিষ্কার কোড:** এমন কিছু লিখবেন না যা অ্যাপকে হ্যাং করে দেয়। যদি ভুল কোড লিখেন, আমাদের **Extension Manager** অটোমেটিক সেটি বন্ধ করে দিবে।
3. **রিমুভাল:** যখন আপনি এক্সটেনশন রিমুভ করবেন, এটি পাকাপাকিভাবে রিমুভ হয়ে যাবে এবং একটি স্মার্ট রিলোড হবে যাতে কোনো মেমরি লিক না থাকে।

---

## ৪. অ্যাপের কোথায় কোথায় আপনি বদলাতে পারবেন? (স্লটস)

- **সাইডবার:** নতুন মেনু বা শর্টকাট বাটন যোগ করা।
- **এডিটর টুলবার:** লেখার স্টাইল বদলানোর বা এআই টুল যোগ করা।
- **নতুন ব্লক:** প্যারাগ্রাফ বা হেডিং এর মতো নিজস্ব কাস্টম ব্লক (যেমন: পোল, চার্ট) বানানো।
- **ডাটা পাইপলাইন:** ডাটা লোড বা সেভ হওয়ার সময় ফিল্টার করা।

এখনই আপনার নিজের একটি টুল বানিয়ে ফেলুন! কোনো হেল্প লাগলে এআই এজেন্টের সাথে কথা বলুন। 🚀
