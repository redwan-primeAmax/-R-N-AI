# Redwan AI Note 📝

A professional, AI-powered note-taking application designed for speed, efficiency, and intelligence.

## 🚀 Features

- **AI-Powered Writing:** Generate, summarize, and enhance your notes using advanced AI models.
- **PWA Support:** Install the app on your device and use it offline.
- **Rich Text Editor:** A clean and powerful editor with support for formatting, task lists, and more.
- **Local Storage:** Your notes are stored locally on your device for maximum privacy.
- **Tools Dashboard:** Built-in tools for word counting, number removal, and text summarization.
- **Smart Search:** Quickly find your notes using a fast indexing system.

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, Framer Motion.
- **AI Integration:** Google Gemini API.
- **Offline Support:** Vite PWA Plugin.
- **Routing:** React Router DOM.

## 📦 Installation

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Create a `.env` file and add your `GEMINI_API_KEY`.
4. Run `npm run dev` to start the development server.

## 📱 Progressive Web App (PWA)

This app is fully PWA-compliant. You can install it on Chrome (Desktop), Android, and iOS (Safari -> Add to Home Screen) to use it like a native application with offline capabilities.
