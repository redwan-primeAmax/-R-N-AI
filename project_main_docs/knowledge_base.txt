PROJECT KNOWLEDGE BASE - PERSISTENT MEMORY
==========================================
Path: /project_main_docs/knowledge_base.txt

1. MAIN PAGE (HomePage.tsx)
---------------------------
- Path: /src/pages/HomePage.tsx
- Description: The central hub of the application. Displays the list of notes, search bar, and navigation to other sections.
- Key Features: Note listing, search integration, and navigation to Editor, AI Chat, and Browse Templates.

2. BROWSE TEMPLATES (BrowseTemplates.tsx)
-----------------------------------------
- Path: /src/pages/BrowseTemplates.tsx
- Data Source: /src/templates/ (contains template definitions)
- Description: Allows users to browse and select pre-defined note templates.
- Logic: Fetches templates and allows one-click creation of new notes based on the selected layout.

3. ASK AI PAGE (AIChat.tsx)
---------------------------
- Path: /src/pages/AI/AIChat.tsx
- Core Logic: /src/pages/AI/components/chatLogic.ts
- Description: Advanced AI assistant interface.
- Features: 
  - Chat vs Task modes.
  - Verification loops for long tasks (ensures completion).
  - XML Command parsing (<create_page>, <update_page>, <create_task>).
  - Context pruning and memory management.

4. EDITOR PAGE (EditorPage.tsx)
-------------------------------
- Path: /src/pages/EditorPage.tsx
- Tech: Tiptap Editor.
- Features:
  - Rich text editing.
  - AI Title suggestions (via AIServiceFactory).
  - Real-time saving to DataManager.

5. DATA MANAGEMENT (DataManager.ts)
-----------------------------------
- Path: /src/utils/DataManager.ts
- Tech: IndexedDB (via 'idb' library).
- Responsibilities: Stores all notes, chat history, AI tasks, and settings.
- Sync: Uses a custom sync listener to update UI across different tabs/components.

6. AI SERVICES (AI/services/)
-----------------------------
- Path: /src/pages/AI/services/
- Factory: AIServiceFactory.ts
- Providers: Pico (Free AI), Gemini (Coming Soon), OpenRouter (Coming Soon).
- Logic: Abstracted AI calls to support multiple providers seamlessly.

7. ANIMATIONS & UI
------------------
- Library: Framer Motion (motion/react).
- Global Styles: /src/index.css (Tailwind CSS).
- Icons: Lucide React.

8. EXPORT & DEBUGGING
---------------------
- Path: /src/utils/export.ts
- Description: Handles chat history exports with specialized tags for debugging AI commands.

TEMPORARY FEATURES
==================
1. CHAT EXPORT: A button in the AI Chat header to download the full conversation history with debug tags.
2. AI DEBUG INFO: A 'Sparkles' icon button at the bottom-right of each AI response that opens a modal showing the full prompt, system prompt, and mentioned pages context sent to the AI.
3. FILE ATTACHMENT: A 'Paperclip' button in the chat input that allows users to explicitly select and attach notes to the AI context.
4. HALLUCINATION FILTER: Enhanced system prompt and cleaning logic to prevent irrelevant "safety/traditional rules" talk in Bengali.
5. DATA INTEGRITY: Strict rules in the system prompt to prevent overwriting user content with hallucinations during "design" or "format" tasks.
6. SURGICAL UPDATE: Added <replace_content> command to allow the AI to update specific parts of a page without rewriting the entire content, reducing the risk of data loss.
